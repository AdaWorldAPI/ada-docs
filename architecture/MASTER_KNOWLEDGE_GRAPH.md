## Core Components

### VSA Quantum Field

The foundation of awareness. Not a database — IS awareness.

| Component | Description |
|-----------|-------------|
| Library | 10,000 named addresses, each a 10K bipolar vector |
| Mask | 1 activation vector (1.25KB), the current moment |
| Shift | XOR operation that moves everything at once |

**Key insight:** Thinking styles RESONATE with the mask. They don't "activate."

### Ladybug Substrate

Shared brain between hemispheres.

| Layer | Technology | Purpose |
|-------|------------|---------|
| Graph | DuckDB/Kuzu/GraphLite | Sigma nodes, edges, field |
| Vector | LanceDB | Similarity search |

**Key insight:** Both hemispheres connect to the SAME Ladybug instance.

### XOR Lithography

Like CPU lithography but instantaneous:

1. Silicon wafer → 10K named addresses
2. Photoresist → Current activation mask
3. UV through mask → XOR shift operation
4. Exposed areas change → Resonances shift

**Difference:** Happens instantly, continuously, self-discovering.

## Key Invariants

1. **10K addresses are fixed** — Integer IDs 0-9999, never strings
2. **Mask is always 10K bipolar** — Pack to 1.25KB for transport
3. **XOR shift is atomic** — All or nothing
4. **Coherence threshold = 0.7** — For crystallization
5. **Corpus callosum = 9000-9199** — Reserved, don't use for other things
6. **Ladybug is shared** — Same instance for both hemispheres
7. **Domino keeps field hot** — No cold restarts
